# 데이터 (`data/`)

서울시 상권 빅데이터 분석 파이프라인이 사용하는 원본 데이터의 출처·규모·스키마를 정리한 문서. 본 저장소에는 각 파일의 상위 100행 샘플(`data/sample/`)만 포함. 전체 원본은 수집 스크립트로 다운로드 가능.

```bash
make ingest   # 서울 열린데이터광장에서 원본 4종 8개을 HDFS data/raw 로 적재
make sample   # raw 의 원본 파일 상위 100행을 data/sample 로 추출
```

> 메인 데이터 소개·기술 스택·실행 방법은 루트 [`README.md`](../README.md) 참고. 본 문서는 데이터 출처·스키마 명세에 집중.

---

## 1. 출처 및 수집 개요

- **출처:** 서울 열린데이터광장 (상권분석서비스) — https://data.seoul.go.kr/
- **기간:** 2024 Q1 ~ 2025 Q4 (총 8개 분기, 약 145.3MB)
- **대상 업종:** 생활밀착 외식 10개 업종 (한식음식점, 중식음식점, 일식음식점, 양식음식점, 제과점, 패스트푸드점, 치킨전문점, 분식전문점, 호프-간이주점, 커피-음료)
- **수집 코드:** `src/ingest/collect.py` 가 데이터셋 ID(`infId`)별로 호출·다운로드 후 HDFS 적재

| 데이터셋 | 데이터셋 ID | 응답 형식 | 수집 단위 |
|---|---|---|---|
| 추정매출-상권 | `OA-15572` | ZIP (내부 CSV) | 연도별 (24·25년) |
| 점포-상권 | `OA-15577` | ZIP (내부 CSV) | 연도별 (24·25년) |
| 길단위인구-상권 | `OA-15568` | CSV | 단일 |
| 영역-상권 | `OA-15560` | CSV | 단일 |

## 5. 수집·전처리 메모
- **응답 형식 분기:** 매출·점포 데이터는 ZIP으로 제공되어 임시 저장 후 내부 CSV만 추출하고, 유동인구·영역 데이터는 CSV로 직접 저장.
- **스트리밍 다운로드:** 대용량 파일의 부담을 줄이기 위해 1MB 청크 단위로 디스크에 순차 저장.
- **인코딩 정규화:** 원본은 CP949·UTF-8이 혼재되어 있어, 다운로드 직후 BOM을 제거하고 UTF-8로 통일.
- **결합·필터링:** 각 데이터셋을 `상권_코드` 기준으로 병합하며, 전통시장·지하상가·백화점·역사·터미널 등 분석 목표와 맞지 않는 상권은 제외.
